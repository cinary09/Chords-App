## 🔴 THE PROBLEM
```
Every route returns 500 error
No error messages in console
Impossible to debug
```

## 🟢 THE SOLUTION
```
app.logger.error(f"500 Error: {error}", exc_info=True)
                  ↑
         One line fixes everything
```

## 🚀 GET STARTED

### 1. Verify dependencies
```bash
pip install -r requirements.txt
```

### 2. Initialize database (if needed)
```bash
flask --app app.py init-db
```

### 3. Start the app
```bash
python run.py
```

### 4. Visit in browser
```
http://127.0.0.1:5000/
```

### 5. If something breaks, run diagnostics
```bash
python test_app.py
```
